# Projeto Front-End React - Tailwind - Next - TypeScript - API

  

Este é um projeto desenvolvido como parte de uma entrevista técnica para uma empresa de desenvolvimento de software. O objetivo foi consumindo uma API fornecida criar um sistema de cadastro de usuários focando na usabilidade e interface para o usuário final.


## Tecnologias Utilizadas

  

- React

- NextJS

- JavaScript

- TypeScript

- JavaScript

- Tailwind CSS

  

## Como Executar o Projeto

  

Siga os passos abaixo para executar o projeto em seu ambiente local:

  

1. Clone o repositório utilizando o seguinte comando:

```sh

git clone https://github.com/Breno-Lima/Challenge-React.git

```

  

2. Abra o terminal, se dirija até a pasta do projeto e execute esse comando:

  ```sh
npm i

```
3. Para rodar o projeto basta rodar esse comando no terminal:
```sh
npm run dev
```

### Para uma melhor experiência fiz o deploy da aplicação na Vercel:
```sh
https://challenge-react-three.vercel.app/
```

1. As senhas deverão ter no mínimo 8 caracteres, uma letra maiúscula, uma letra minúscula e um número.
